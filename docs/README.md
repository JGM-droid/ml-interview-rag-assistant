# Knowledge Base Documents

This folder contains the study corpus used by the RAG pipeline in `ml_interview_rag_assistant.ipynb`.

## Document Index

- `01_ml_fundamentals.md`: ML foundations and workflow framing.
- `02_data_preprocessing.md`: cleaning, missing data, scaling, and encoding basics.
- `03_feature_enginering_encoding.md`: feature engineering patterns and encoding tradeoffs.
- `04_supervised_learning.md`: core supervised learning algorithms and usage context.
- `05_model_evaluation.md`: metrics, confusion matrix concepts, and evaluation practice.
- `06_nueral_networks_deep_learning.md`: neural network and deep learning fundamentals.
- `07_computer_vision.md`: computer vision foundations and CNN concepts.
- `08_nlp_rag.md`: NLP basics, embeddings, and RAG concepts.

## Design Choice

The repository stores domain knowledge as standalone Markdown files rather than embedding all content directly in the notebook. This keeps source content easier to review, update, and expand.
